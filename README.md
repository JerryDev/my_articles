# About

Person archives.

## 使用方法

### 1. 安装依赖

    yarn install
    
### 2. 启动服务

    yarn hexo server
    
### 3. 生成网站

    yarn hexo generate
    
### 4. 推送到github

    yarn hexo clean && yarn hexo deploy
